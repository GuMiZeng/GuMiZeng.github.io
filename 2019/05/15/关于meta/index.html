html标签： lang属性

zh和 zh-CN已经属于废弃用法了；
lang = zh-cmn-Hans 简体中文
lang = zh-cmn-hant 繁体中文
lang = en 英文
一般不会加上区域代码 例如：zh-cmn-Hans-CN

http-equiv属性

相当于HTTP的文件头，向浏览器传回信息

  1. Expires 期限 : 设定网页到期时间，过期重新传输 GMT时间格式

  2. Pragma（cache模式）: 禁止浏览器从本地计算机的缓存中访问页面

  3. Refresh (刷新): 自动刷新并指向新的页面

  4. Set-Cookie 网页过期 存盘的cookie删除 GMT时间格式

  5. Window-target (显示窗口的设定) 防止别人在框架里调用自己的页面

  6. content-Type （显示字符集的设定）

  7. content-Language（显示语言设定）

关于ios iPad iPhone

一般的4.7~5寸的设备viewport在360px；大部分5.5寸的在400px。
<meta name="viewport" content="width=375">
<!--iPhone 357px 尺寸-->
<meta name="viewport" content="width=414">
<!--iPhone 414px 尺寸-->

<meta name="apple-mobile-web-app-title" content="标题">
<!-- 添加到主屏后的标题（iOS 6 新增） -->
<meta name="apple-mobile-web-app-capable" content="yes" />
<!-- 是否启用 WebApp 全屏模式 -->
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent" />
<!--
设置状态栏的背景颜色
只有在 “apple-mobile-web-app-capable” content=”yes”
也就是启用全屏模式时生效

content 参数：
default 默认值。
black 状态栏背景是黑色。
black-translucent 状态栏背景是黑色半透明。 如果设置为 default 或 black ,网页内容从状态栏底部开始。 如果设置为 black-translucent ,网页内容充满整个屏幕，顶部会被状态栏遮挡
-->
<meta name="format-detection" content="telephone=no" />
<!-- 禁止数字识自动别为电话号码 -->
